# Restrict numeric data to the interval \[lower, upper\]

Restrict numeric data to the interval \[lower, upper\]

## Usage

``` r
snap(x, lower, upper, ...)
```

## Arguments

- x:

  a numeric vector

- lower:

  number, the lower bound

- upper:

  number, the upper bound

- ...:

  unused

## Value

An object of the same type as `x`
